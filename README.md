# TodoList
# Aplikasi To-Do List
## Deskripsi
Aplikasi sederhana berbasis PHP untuk mencatat tugas harian.
## Fitur
- Tambah tugas
- Tandai tugas selesai
- Hapus tugas
## Struktur Folder
-'index.php'-halaman utama
-'functions.php'- fungsi
-'style.css' - css
## Cara Menjalankan
1. Salin folder ke 'htdocs/
2. Jalankan XAMPP dan buka di`http://loealhost/todolist
